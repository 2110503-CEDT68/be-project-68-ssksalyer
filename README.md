
# Dentist Booking System

## Features

- User registration with name, telephone, email, and password
- User login/logout
- Registered users can book ONE session with a preferred dentist (date and dentist)
- Dentist information includes name, years of experience, and area of expertise
- Users can view, edit, and delete their booking
- Admin can view, edit, and delete any booking

## API Endpoints

- `/api/v1/auth/register` - Register user
- `/api/v1/auth/login` - Login user
- `/api/v1/auth/logout` - Logout user
- `/api/v1/dentists` - List dentists
- `/api/v1/appointments` - Manage bookings
